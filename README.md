# Het-19-9-Systeem-N-19A-9B

Het 19-9 Systeem

N = 19A + 9B

Een systematische analyse van het lineaire diofantische systeem N = 19A + 9B met A, B ≥ 0.

Kernobservaties

- Representatieformule: exacte afleiding van R(N), het aantal oplossingen voor elk N
- Digitale wortel als startwaarde: A₀ = dr(N) door 19 ≡ 1 (mod 9)
- Structuur van oplossingen: stapvector (+9 in A, −19 in B) en familiegrenzen 171k + 144
- Ankerfamilie: 1675–1683 als scharnierpunt waar alle digitale wortels 1–9 convergeren
- Fibonacci-verbinding: F(18) = 2584 = 19 × 136 exact (B = 0, Pisano-punt)
- Convergentie-eigenschappen: quasi-eigenvector-gedrag in Newton-Raphson voor φ
